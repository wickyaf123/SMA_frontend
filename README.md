# PermitScraper.ai -- Frontend

React + Vite + TypeScript frontend for the PermitScraper.ai permit lead generation platform.

## Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query

## Scripts

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Production build
npm run build

# Preview production build locally
npm run preview
```
